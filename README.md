# bpc-akr-projekt


## Required packages

### Installing

- Be inside active venv
- Run: `pip3 freeze > requirements.txt`

### Dumping

- Be inside active venv
- Run: `pip install -r requirements.txt`


## VENV

### Activation

- Be inside `client` or `server` directory
- Run: `python3 -m venv .venv`

### Deactivation

- Close terminal
